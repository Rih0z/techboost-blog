## はじめに

Geekly（ギークリー）はIT・Web・ゲーム業界特化の転職エージェントだ。

「ゲーム業界への転職といえばGeekly」という評判がある一方、Web系エンジニアにも強い。本記事では、Geeklyのサービス内容・強み・デメリットを詳しく解説し、どのエンジニアに向いているかを明確にする。

---

## Geekly の基本情報（2026年版）

| 項目 | 内容 |
|------|------|
| 運営会社 | 株式会社Geekly |
| 設立 | 2012年 |
| 対象職種 | エンジニア・デザイナー・ゲーム開発者・PM |
| 求人数 | 約10,000件以上 |
| 対象年齢 | 20代〜30代がメイン |
| 業界特化 | IT・Web・ゲーム・エンタメ |
| 対応エリア | 東京・大阪・愛知・福岡 |

## Geekly の強み

### 1. ゲーム業界転職に圧倒的な強み

Geekly最大の特徴は**ゲーム・エンタメ業界への転職支援力**だ。

大手ゲーム会社（コナミ・カプコン・セガ等）からモバイルゲーム会社（ガンホー・DeNA等）まで、ゲーム業界の求人を豊富に保有している。

```
Geeklyが特に強いゲーム系企業（例）:
- 任天堂・ソニーインタラクティブエンタテインメント等の大手
- スクウェア・エニックス・バンダイナムコ等のRPGメーカー
- mixi・gloops等のモバイルゲーム会社
- Cygames・Cygames系列スタジオ
```

### 2. Web系・スタートアップ求人も充実

ゲーム業界だけでなく、**Web系企業・スタートアップ・メガベンチャー**の求人も豊富。

- React/Vue/TypeScript等のモダンWeb開発
- バックエンド（Go/Python/Ruby等）
- データエンジニア・ML エンジニア
- テックリード・エンジニアリングマネージャー

### 3. 担当者のIT専門知識

GeeklyのキャリアアドバイザーはIT業界出身者が多く、**技術的な会話ができる担当者**が多い。

```
担当者が対応できること:
- 「TypeScriptのジェネリクスが得意です」という強みの言語化
- ゲームエンジン（Unity/Unreal Engine）経験者への適切なアドバイス
- スタートアップ vs 大手ゲーム会社のカルチャー違いの説明
```

### 4. 非公開求人の多さ

Geeklyで紹介される求人の一部は**一般公開されていない非公開求人**。求人サイトに載っていない好条件の求人にアクセスできる点は大きなメリットだ。

### 5. 内定後のサポートが手厚い

内定後の給与交渉・退職交渉のサポートも充実している。

```
内定後サポート例:
- 「年収600万を提示されたが700万希望」→ 担当者が企業に交渉
- 現職の退職タイミングに合わせた入社調整
- 複数内定時の比較・選択のアドバイス
```

## Geekly のデメリット

### 1. 関東圏外は求人が限られる

Geeklyは**東京中心のサービス**。大阪・福岡にも対応しているが、地方在住・完全リモート希望者は求人選択肢が少なくなる場合がある。

### 2. 未経験・経験浅い場合は対応が限られる

IT業界未経験者の転職には対応していない。**最低でもITの現職経験が1年以上**ないと求人紹介が難しいケースが多い。

### 3. 登録後の連絡頻度が高い場合がある

一部の口コミで「登録後に連絡が多かった」という声がある。転職時期が先でも登録はできるが、担当者からの連絡頻度が合わない場合は率直に伝えることが重要。

## Geekly の利用手順

### Step 1: 登録

1. 公式サイトで氏名・メール・現職情報を入力
2. 登録後2〜3営業日でキャリアアドバイザーから連絡

### Step 2: カウンセリング（オンラインまたは電話）

```
ヒアリング内容:
- 現在の技術スタック・役割
- 転職理由・転職時期
- 志望業界（ゲーム/Web/その他）
- 希望年収・勤務形態
```

### Step 3: 求人紹介・応募

カウンセリング後、担当者から求人が届く。

```
応募の流れ:
1. 担当者からの求人リスト確認
2. 興味のある求人を担当者に伝える
3. 担当者が企業に書類送付
4. 書類通過後、面接日程を調整
```

### Step 4: 面接 → 内定 → 入社

面接対策セッションも依頼可能。

## レバテックキャリア vs Geekly どちらを使うべきか

| 状況 | 推奨 |
|------|------|
| ゲーム・エンタメ業界志望 | **Geekly** |
| Web系・SaaS企業志望 | **レバテックキャリア** |
| 年収600万円以上を狙う | **レバテックキャリア**（求人数が多い） |
| 東京近郊 かつ IT経験あり | **両方併用** が最も効果的 |
| 経験浅い（1〜2年） | **Geekly**（若手向けサポートあり） |

**結論**: どちらか一方でなく、**両方に登録して良い求人・良い担当者を見つけること**が転職成功の近道。

## 転職成功事例（ケーススタディ）

### ケース1: Unity開発者 → 大手ゲームスタジオ

**プロフィール**: Unity/C# 経験3年（モバイルゲーム中小企業）、年収400万
**転職後**: 大手ゲームスタジオのUnityエンジニア、年収580万

```
Geeklyが行ったこと:
- モバイルゲームの最適化経験を「描画パフォーマンス改善」として整理
- 大手スタジオの技術面接（グラフィクスプログラミングの問題）の傾向を提供
- 年収交渉で550万→580万に引き上げ
```

### ケース2: 受託フロントエンド → ゲーム企業のWeb部門

**プロフィール**: Vue/React 経験2年（受託開発会社）、年収350万
**転職後**: ゲーム企業の自社サービス部門フロントエンド、年収480万

```
Geeklyが行ったこと:
- ゲーム企業が求める「ユーザー体験への感度」を面接でアピールする方法を指導
- 自社サービス（安定性・課題の大きさ）のメリットを整理
- 年収大幅アップの根拠（自社サービス会社への転職プレミアム）を説明
```

## Geeklyを最大活用するコツ

### 1. ゲーム業界への志望度を明確に伝える

「ゲームが好きで、ゲーム業界で働きたい」という動機は Geekly のアドバイザーに刺さりやすい。ゲームへの熱意を率直に伝えることが大切。

### 2. 転職軸を3つ整理しておく

```
整理しておく転職軸の例:
1. 事業フェーズ: 「グロース期のスタートアップよりも安定した大手希望」
2. 技術: 「Unity/C++のゲームエンジン開発がやりたい」
3. 年収: 「現状350万から500万以上へ」
```

### 3. 複数エージェントと並行する

Geeklyに加えて、**レバテックキャリア** や **転職ドラフト**（スカウト型）も並行利用すると、求人の比較・選択肢が広がる。

## まとめ

Geeklyは**ゲーム・エンタメ業界への転職を目指すITエンジニアに最も向いているエージェント**だ。

Web系・スタートアップにも十分な求人があるため、IT業界全般で転職を考えるエンジニアにも使えるサービス。

| 評価項目 | 評価 | 理由 |
|---------|------|------|
| ゲーム業界求人 | ★★★★★ | 業界最多クラス |
| Web系求人 | ★★★★☆ | 充実しているが求人数はレバテックが多い |
| 担当者専門性 | ★★★★☆ | IT業界出身者多数 |
| サポート充実度 | ★★★★☆ | 面接〜内定後まで手厚い |
| 地方・リモート | ★★★☆☆ | 東京中心で地方は限定的 |

IT・ゲーム業界でのキャリアアップを目指すエンジニアは、まず無料登録して担当者と話してみることをおすすめする。
