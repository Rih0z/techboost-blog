## はじめに

確定申告の時期、「どこまで経費にできるか？」は多くのフリーランスエンジニア・副業エンジニアが悩むテーマだ。

この記事では、エンジニアが計上できる経費をカテゴリ別に50項目まとめ、**具体的な計上方法と注意点**を解説する。

> **免責事項**: 本記事は一般的な情報提供を目的としています。個別の税務判断は税理士・会計士に相談してください。

---

## 経費として計上できる条件

経費として認められるためには、**事業に関連していること**が条件だ。

| 区分 | 説明 | 例 |
|------|------|-----|
| **全額経費** | 仕事100%のために使った費用 | 業務専用のドメイン代、クラウドサービス |
| **按分経費** | 仕事と私生活で共用している費用 | 家賃（在宅勤務）、光熱費、通信費 |
| **不可** | 事業関連性のないもの | 純粋な趣味・娯楽費用 |

| # | 経費項目 | 計上条件 | 注意点 |
|---|---------|---------|------|
| 1 | **開発用PC・MacBook** | 業務用なら全額（10万円未満で即時償却） | 10万円以上は減価償却（4年） |
| 2 | **外付けモニター** | 業務用 → 全額 | 私用兼用なら按分 |
| 3 | **メカニカルキーボード** | 業務用 → 全額 | 同上 |
| 4 | **マウス・トラックパッド** | 業務用 → 全額 | 同上 |
| 5 | **USBハブ・ドッキングステーション** | 業務用 → 全額 | - |
| 6 | **Webカメラ** | 業務ミーティング用 → 全額 | - |
| 7 | **ヘッドセット・マイク** | 業務ミーティング用 → 全額 | - |
| 8 | **iPad（作業補助用）** | 業務使用割合で按分 | 私用がメインなら難しい |
| 9 | **ルーター・LANケーブル** | 業務用ネット環境整備 → 全額〜按分 | - |
| 10 | **NAS（バックアップ用）** | 業務データ保管用 → 全額 | - |

```python
# PC減価償却の計算例（定額法）
pc_price = 150000  # 15万円のMacBook
useful_life = 4    # 法定耐用年数4年
annual_depreciation = pc_price / useful_life
print(f"年間減価償却費: {annual_depreciation:,.0f}円")
# 出力: 年間減価償却費: 37,500円
```

### カテゴリ3: 通信費（3項目）

| # | 経費項目 | 計上条件 | 按分目安 |
|---|---------|---------|--------|
| 23 | **インターネット回線費** | 業務使用割合で按分 | 在宅50〜70%が一般的 |
| 24 | **携帯電話代（スマホ）** | 業務使用割合で按分 | 業務利用50%なら50%計上 |
| 25 | **モバイルWi-Fi** | 業務での利用比率で按分 | 外出先での業務が多ければ高率 |

```python
# 通信費の按分計算例
monthly_internet = 5500  # 月5,500円の光回線
work_ratio = 0.6         # 業務利用60%

deductible = monthly_internet * work_ratio
annual_deductible = deductible * 12
print(f"月間経費計上額: {deductible:,.0f}円")
print(f"年間経費計上額: {annual_deductible:,.0f}円")
# 月間経費計上額: 3,300円
# 年間経費計上額: 39,600円
```

### カテゴリ4: 家賃・光熱費（在宅勤務）（3項目）

| # | 経費項目 | 計上条件 | 按分計算方法 |
|---|---------|---------|-------------|
| 26 | **家賃（自宅兼事務所）** | 事務所として使う部屋の割合 | 間取り比率で計算 |
| 27 | **電気代** | 業務時間・使用機器割合 | 使用時間比率で計算 |
| 28 | **水道代** | 事務所使用分（少額） | 通常は微細で計上しないことも |

```python
# 家賃按分の計算例
monthly_rent = 120000  # 家賃12万円
total_area = 60        # 総床面積60平方メートル
work_area = 12         # 仕事部屋12平方メートル

area_ratio = work_area / total_area        # 面積按分: 20%
work_time_ratio = 8 / 24                   # 時間按分: 33%

# 面積と時間の積で按分
combined_ratio = area_ratio * work_time_ratio
monthly_deductible = monthly_rent * combined_ratio

print(f"按分割合: {combined_ratio:.1%}")
print(f"月間経費: {monthly_deductible:,.0f}円")
print(f"年間経費: {monthly_deductible * 12:,.0f}円")
# 按分割合: 6.7%
# 月間経費: 8,000円
# 年間経費: 96,000円
```

> **注意**: 家賃の按分計上は、**賃貸借契約で「事務所使用」が禁止されていない**こと、**実際に業務で使っていること**が条件。持ち家の場合は税理士に確認を。

## 按分計算の実践例

### 在宅エンジニアの年間経費計算シート

```python
# 在宅フリーランスエンジニアの年間経費試算
expenses = {
    # PC・機器
    "MacBook Pro (15万 / 4年減価償却)": 150000 / 4,
    "外付けモニター2台": 60000,
    "キーボード・マウス": 25000,

    # ソフトウェア
    "GitHub Pro": 1100 * 12,
    "JetBrains": 27000,
    "Claude Pro": 3000 * 12,
    "AWS利用料": 5000 * 12,

    # 通信費（按分60%）
    "インターネット回線": 5500 * 12 * 0.6,
    "スマホ代": 8000 * 12 * 0.5,

    # 家賃（按分10%）
    "家賃按分": 120000 * 12 * 0.10,

    # 書籍・学習
    "技術書・Udemy": 80000,
    "AWS資格受験料": 15000 * 2,

    # その他
    "コワーキングスペース": 20000,
    "名刺・事務用品": 10000,
    "会計ソフト(freee)": 3828 * 12,
}

total = sum(expenses.values())
print(f"年間経費合計: {total:,.0f}円")
print(f"所得税率20%で節税額: {total * 0.2:,.0f}円（概算）")
for item, amount in sorted(expenses.items(), key=lambda x: -x[1]):
    print(f"  {item}: {amount:,.0f}円")
```

## 経費を証明するための記録管理

### 領収書・レシートの保存

```
デジタル保存のルール（2023年電子帳簿保存法改正対応）:
- スキャナ保存 or スマホ撮影でOK（一定条件あり）
- freee / マネーフォワード の自動取込機能が便利
- クレジットカード明細は自動で連携可能

保存期間: 青色申告の場合7年間（白色は5年）
```

### 按分の根拠を記録する

```
記録すべきこと:
1. 按分割合の根拠（家賃ならメジャーで測った図面、通信費なら業務時間記録）
2. 購入目的（何のために買ったか）
3. 業務での実際の使用履歴（GitHubのコミット履歴、作業ログ等が証拠になる）
```

## 青色申告にすることで経費以外の節税効果

| 制度 | 節税効果 | 条件 |
|------|---------|------|
| **青色申告特別控除** | 最大65万円控除 | e-Tax + 複式簿記 |
| **少額減価償却資産** | 30万円未満を即時全額経費化 | 年間合計300万円まで |
| **赤字の繰越控除** | 損失を3年間繰り越せる | - |
| **専従者給与** | 配偶者等に支払う給与を経費化 | 届出が必要 |

## よくある誤解と注意点

### 誤解1: 「家族との外食は全部経費にできる」

→ **不可**。家族との外食は「接待交際費」になるには相手が取引先等である必要がある。家族間の外食は経費にならない。

### 誤解2: 「スマホを買い替えたら全額経費」

→ **按分が必要**。業務使用割合（50〜80%等）のみ経費計上可能。

### 誤解3: 「趣味のゲームはUI研究のため経費」

→ **基本的に不可**。「業務関連性」が明確でない費用は税務調査で否認されるリスクが高い。

### 注意: 経費の過大計上は税務調査のリスク

売上と比較して経費率が異常に高い場合（例: 売上500万で経費400万 = 経費率80%）は税務調査の対象になりやすい。適正な按分を心がけること。

## まとめ

エンジニアが確定申告で計上できる経費は、PC代から通信費・クラウドサービス費用まで多岐にわたる。

### 節税の優先順位

1. **最優先**: ソフトウェア・クラウドサービス費用（全額・証明しやすい）
2. **重要**: PC・機器代（金額が大きい）
3. **有効**: 通信費・光熱費の按分（継続的に積み上がる）
4. **要注意**: 家賃の按分（根拠を整備する必要がある）

freeeやマネーフォワードで記帳を自動化しつつ、不明な点は早めに税理士に確認することが、税務リスクを下げながら節税効果を最大化するコツだ。
