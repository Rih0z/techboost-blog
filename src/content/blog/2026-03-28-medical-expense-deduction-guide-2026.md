## 医療費控除とは

### 基本的な仕組み

医療費控除とは、1年間（1月1日〜12月31日）に支払った医療費が一定額を超えた場合に、超えた金額を所得から差し引ける制度だ。

```
医療費控除額 = 実際に支払った医療費の合計 - 保険金などで補填された金額 - 10万円（または所得の5%）

※控除限度額: 200万円まで
```

**ポイント:**
- 対象は本人だけでなく、生計を一にする家族（配偶者・子ども・親など）の医療費も合算できる
- 給与所得者と同様に、個人事業主も確定申告で申請できる
- 医療費控除は所得控除のため、所得税・住民税の両方で節税効果がある

### 医療費控除の計算式（詳細）

```
医療費控除の計算ステップ:

Step 1: 年間医療費合計を集計する
  ├─ 本人分
  ├─ 配偶者分
  └─ 生計を一にする家族分（合算）

Step 2: 保険金補填額を差し引く
  ├─ 健康保険の高額療養費
  ├─ 民間医療保険の入院給付金
  └─ 出産育児一時金 など

Step 3: 控除計算
  ├─ 所得が200万円以上: 10万円を差し引く
  └─ 所得が200万円未満: 所得の5%を差し引く

Step 4: 控除上限: 200万円まで
```

## 医療費控除の対象となる費用

### 対象になるもの（主要項目）

| カテゴリ | 具体例 |
|---------|--------|
| 診療・治療費 | 病院・クリニックの診察費、手術費 |
| 入院費 | 入院中の部屋代・食事代（差額ベッド代除く） |
| 薬代 | 処方薬（市販薬は条件付き） |
| 歯科治療 | むし歯治療、入れ歯・差し歯 |
| 通院交通費 | 公共交通機関の運賃（バス・電車） |
| 出産費用 | 妊娠・出産に関する費用 |
| 介護費用 | 介護サービス費（一部） |
| 松葉杖・義歯 | 医師の指示による補助器具購入費 |

### 対象にならないもの

| 項目 | 理由 |
|------|------|
| 健康診断・人間ドック費用 | 治療ではないため（異常発見→治療は対象） |
| 美容整形 | 治療目的でないため |
| 差額ベッド代 | 選択的なもの |
| 自家用車の通院ガソリン代 | 公共交通機関のみ対象 |
| 視力矯正レーシック | 原則対象外（疾患治療は別） |
| 栄養補助食品・サプリメント | 医薬品ではないため |

### 市販薬（OTC医薬品）が対象になるケース

2017年以降、一定の市販薬（スイッチOTC医薬品）は医療費控除の対象となる。ただし「セルフメディケーション税制」として通常の医療費控除とは別制度で、**どちらか一方しか使えない**。

```
市販薬（OTC医薬品）の医療費控除適用条件:
  ✓ 「スイッチOTC医薬品」として指定されている製品
  ✓ 医師の処方なしで購入できるが、もともと処方薬だったもの
  例: ロキソニンS、ガスター10、アレグラFX など
```

## 具体的な計算例

### 計算例1: 単身フリーランスエンジニア

```
前提条件:
  年収: 600万円（事業所得）
  経費: 150万円
  事業所得: 450万円

年間医療費:
  ├─ 歯科治療（インプラント）: 15万円
  ├─ 風邪・胃腸炎の診察+薬代: 3万円
  ├─ 通院交通費（電車）: 8,000円
  └─ 合計: 18万8,000円

保険金補填なし（民間医療保険未加入）

医療費控除額:
  188,000円 - 100,000円 = 88,000円

節税効果（所得税+住民税）:
  ├─ 所得税率20%（450万円の場合）: 88,000円 × 20% = 17,600円
  └─ 住民税率10%: 88,000円 × 10% = 8,800円
  合計節税額: 約26,400円
```

### 計算例2: 家族持ちフリーランス

```
前提条件:
  事業所得: 500万円
  家族構成: 本人 + 配偶者（専業主婦）+ 子ども2人

年間医療費（家族合算）:
  ├─ 本人（健康診断で異常発見→胃カメラ治療）: 5万円
  ├─ 配偶者（婦人科・産科）: 35万円（出産費用含む）
  │   └─ 出産育児一時金50万円補填
  ├─ 子ども1（歯科矯正治療目的）: 20万円
  └─ 子ども2（アトピー治療・通院費）: 8万円
  合計: 68万円

保険金補填:
  └─ 出産育児一時金: 50万円

対象医療費: 68万円 - 50万円 = 18万円

医療費控除額: 18万円 - 10万円 = 8万円

節税効果:
  ├─ 所得税20%: 8万円 × 20% = 16,000円
  └─ 住民税10%: 8万円 × 10% = 8,000円
  合計節税額: 約24,000円
```

### 歯科矯正の医療費控除（よく聞かれる質問）

```
歯科矯正の医療費控除適用判断:
  ✓ 子どもの歯並び矯正: 基本的に対象（発育上の治療）
  ✓ 医師の診断書がある歯科矯正: 対象
  △ 成人の審美目的矯正: 原則対象外
  ✓ かみ合わせ・口腔機能改善目的の矯正: 対象
```

## セルフメディケーション税制との比較

### どちらを選ぶべきか

| 比較項目 | 通常の医療費控除 | セルフメディケーション税制 |
|---------|----------------|------------------------|
| 対象 | 病院・薬局全般 | 指定のスイッチOTC医薬品のみ |
| 足切り額 | 10万円（所得5%） | 1万2,000円 |
| 上限額 | 200万円 | 8万8,000円 |
| 適用条件 | 特定健診・予防接種などの受診 | 必要 |
| 選択 | どちらか一方のみ | どちらか一方のみ |

### 選択の目安

```
セルフメディケーション税制が有利なケース:
  ✓ 年間の病院・医療費が10万円未満
  ✓ スイッチOTC医薬品を年間1万2,000円以上購入している
  ✓ 健康診断・予防接種・特定健診を受けている

通常の医療費控除が有利なケース:
  ✓ 年間の医療費合計が10万円を大幅に超える
  ✓ 入院・手術・出産などの高額医療があった
  ✓ 家族全員の医療費を合算すると10万円を超える
```

## 申請に必要な書類と手順

### 必要書類チェックリスト

```
□ 確定申告書（マイナポータル連携または手書き）
□ 医療費控除の明細書（集計表）
□ 医療費の領収書（5年間保存義務。税務署提出は不要になったが保存必須）
□ 医療費通知（健康保険組合から届く「医療費のお知らせ」）
   └─ ある場合は集計を省略できる
□ 保険金・給付金の支払通知書（補填があった場合）
□ 交通費の記録（日付・目的・金額のメモ）
```

### 申請の手順（e-Tax利用）

**Step 1: 医療費の集計**

```
集計方法:
1. 医療費の領収書を月別・家族別に整理
2. Excelまたはスプレッドシートで集計
3. 「医療費の明細書」フォームに入力

集計表のサンプル（Excel/Googleスプレッドシート）:

**Step 2: 国税庁の確定申告書作成コーナーで入力**

```
操作手順:

3. 「所得控除」→「医療費控除」を選択

## よくある落とし穴とQ&A

### Q1: 健康診断費用は対象になるか？

**A:** 原則として対象外。ただし、健康診断で異常が発見されて治療を受けた場合は、健診費用も遡って医療費控除の対象となる。

### Q2: 市販薬（風邪薬・痛み止め）は対象になるか？

**A:** 通常の市販薬は医療費控除の対象外。ただし「スイッチOTC医薬品」に指定された製品を年間1万2,000円以上購入した場合は「セルフメディケーション税制」が使える（通常の医療費控除とどちらか一方を選択）。

### Q3: 遠方の専門病院への新幹線代は対象になるか？

**A:** 公共交通機関であれば対象となるが、「近くに同等の医療機関がない」場合など合理的な理由が必要。領収書は保管しておくこと。

### Q4: 介護費用は対象になるか？

**A:** 対象になるものとならないものがある。対象になるのは医療系サービス（訪問看護・訪問リハビリ・通所リハビリなど）。対象外は福祉系サービス（通所介護・訪問介護の生活援助など）。

### Q5: 海外での医療費は対象になるか？

**A:** 対象となる。ただし、外貨で支払った場合は円換算が必要（支払日の為替レートを使用）。領収書（日本語訳も用意）を保管しておくこと。

## まとめ

医療費控除は、個人事業主・フリーランスでも確定申告で必ず活用すべき所得控除の一つだ。

**重要ポイントのまとめ:**

1. **10万円を超えた医療費が控除の対象**（所得200万円未満は所得の5%）
2. **家族全員の医療費を合算**できる（生計を一にする家族）
3. **補填された保険金は差し引く**（高額療養費・医療保険給付金など）
4. **セルフメディケーション税制とどちらか一方**しか選べない
5. **領収書は5年間保管**（税務署への提出は不要になったが保存は必須）
6. **確定申告ソフトを使えば**集計・申告が大幅に楽になる

医療費の記録は年間を通じてこまめに行うのが最大のコツ。来年の確定申告に向けて、今日からスプレッドシートやアプリで記録を始めよう。

個別の判断に迷う場合は、税理士・会計士へ相談することをお勧めする。
