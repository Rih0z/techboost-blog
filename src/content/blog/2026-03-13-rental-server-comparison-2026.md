## WordPress向けサーバー選びの重要ポイント

### 1. 表示速度（Core Web Vitals対策）

Googleのランキングアルゴリズムに組み込まれたCore Web Vitalsでは、サーバーの応答速度（TTFB: Time to First Byte）が重要な指標となる。

2026年現在、TTFBの目安は**200ms以下**が推奨。XServerとConoHa WINGはこの基準を安定的にクリアしている。

```
# WordPress表示速度の簡易チェック（curl使用）
curl -o /dev/null -s -w "TTFB: %{time_starttransfer}s\nTotal: %{time_total}s\n" https://your-site.com/
```

### 2. WordPress対応機能の充実度

- **自動バックアップ**: 毎日自動でデータをバックアップしてくれるか
- **SSL証明書**: Let's Encryptが無料で使えるか
- **PHPバージョン切替**: PHP8.x系への対応状況
- **データベース**: MySQLのバージョンと数の制限

### 3. サポート品質

初心者にとって、困ったときに頼れるサポートは非常に重要。

| サービス | サポート手段 | 対応時間 |
|---------|------------|--------|
| XServer | メール・電話 | 平日10-18時（電話） |
| ConoHa WING | チャット・メール | 24時間 |
| さくら | メール | 24時間（返信は翌日以降） |
| ロリポップ | チャット・メール | 平日10-18時 |
| お名前.com | チャット・メール・電話 | 24時間 |

### 4. 料金と契約期間

多くのサービスは**長期契約ほど月額が安くなる**仕組みを採用。

- 36ヶ月契約: 最も月額が安い（更新時は通常料金に戻ることが多い）
- 12ヶ月契約: バランスが取れた選択肢
- 月払い: 試用には良いが割高

**注意**: 初回割引キャンペーンを利用する場合、**更新時の料金**も必ず確認すること。

## よくある質問（FAQ）

**Q: 最初は安いサーバーから始めて、後で移行できますか？**

A: 可能ですが、WordPressの移行作業は初心者には手間がかかります。最初から用途に合ったサーバーを選ぶほうが長期的にはコスパが良いです。

**Q: 無料SSLとは何ですか？**

A: サイトのURLを「http://」から「https://」に変更するための証明書です。Googleのセキュリティ評価に影響し、SEO上も重要です。現在はほぼ全サービスで無料提供されています。

**Q: サーバーの乗り換えキャンペーンはありますか？**

A: XServerとConoHa WINGは定期的に乗り換えキャンペーンを実施しています。公式サイトのキャンペーン情報を確認してください。

**Q: WordPressに最適なPHPバージョンは？**

A: 2026年現在はPHP 8.2以上が推奨されています。選ぶサーバーがPHPバージョンを手動で切り替えられるか確認してください。
